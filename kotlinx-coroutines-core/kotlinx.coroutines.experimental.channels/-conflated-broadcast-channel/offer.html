offer

fun offer(element: E): Boolean (source)

Sends the value to all subscribed receives and stores this value as the most recent state for future subscribers. This implementation always returns true.

It throws ClosedSendChannelException if the channel isClosedForSend normally. It throws the original close cause exception if the channel has failed.
