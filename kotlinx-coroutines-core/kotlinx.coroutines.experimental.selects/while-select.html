whileSelect

suspend fun whileSelect(
    builder: SelectBuilder<Boolean>.() -> Unit
): Unit (source)

Loops while select expression returns true.

The statement of the form:

whileSelect {
    /*body*/
}

is a shortcut for:

while(select<Boolean> {
   /*body*/
}) {}
