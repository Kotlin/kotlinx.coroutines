@ExperimentalCoroutinesApi()
fun TestCoroutineScope(context: CoroutineContext = EmptyCoroutineContext): TestCoroutineScope
Content copied to clipboard

A scope which provides detailed control over the execution of coroutines for tests.

If the provided context does not provide a ContinuationInterceptor (Dispatcher) or CoroutineExceptionHandler, the scope adds TestCoroutineDispatcher and TestCoroutineExceptionHandler automatically.

Parameters

context

an optional context that MAY provide UncaughtExceptionCaptor and/or DelayController
