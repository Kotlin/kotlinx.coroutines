rxMaybe

fun <T> rxMaybe(
    context: CoroutineContext,
    block: suspend CoroutineScope.() -> T?
): Maybe<T> (source)

Creates cold maybe that will run a given block in a coroutine. Every time the returned observable is subscribed, it starts a new coroutine in the specified context. Coroutine returns a single, possibly null value. Unsubscribing cancels running coroutine.

Coroutine action Signal to subscriber
Returns a non-null value onSuccess
Returns a null onComplete
Failure with exception or unsubscribe onError
